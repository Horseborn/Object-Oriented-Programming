What is object oriented programming?

Object oriented programming, or "OOP" is a programming paradigm centered arounds objects rather than functions.
A "paradigm" can be explained by thinking of it as a thought pattern or as a distinct set of concepts. Another example of this is MVC (or Model View Controller). MVC is also a programming paradigm.
